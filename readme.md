This is a npm test package, will be deleted soon.

# Installation
`npm install rvshadowwizard --save`

then...

```
import { rvshadowwizard } from 'rvshadowwizard'

rvshadowwizard({
    shadow_type: 'soft',
    padding: false,
});
```

##Options
* *shadow_type* - _hard | soft_ (Defaults to soft)
* *padding* _boolean_ (Defaults to false)
